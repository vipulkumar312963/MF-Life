# 📱 MF Life - All-in-One Utility App

MF Life is a powerful Android application designed to simplify everyday life by combining multiple tools into one platform.

---

## 🚀 Features

### 💰 Budget & Finance

* Expense Tracker
* Monthly & Annual Budget
* Income vs Expenses Analysis
* Debt Payoff Planner

### 👥 Bill Splitter

* Split bills with friends/groups
* Track balances
* Group management

### ⏰ Bill Reminders

* Add and manage bill reminders
* Notifications for upcoming bills

### 💡 Financial Calculators

* EMI Calculator
* SIP Calculator
* Loan Eligibility
* Retirement Planner
* Tax Calculator
* Net Worth Calculator

### 🎯 Savings Goals

* Create and track savings goals
* Monitor progress

### ⭐ Favorites

* Save frequently used tools

---

## 🛠️ Tech Stack

* Kotlin
* Android SDK
* MVVM Architecture
* Room Database

---

## 📂 Project Structure

* `app/` → Main Android application
* `budget/` → Budget management features
* `billSplitter/` → Group expense splitting
* `financial/` → Financial calculators
* `reminder/` → Bill reminder system
